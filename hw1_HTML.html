    1. Create an empty directory named <uni>_hw1. Initialize the directory into a Git repo.
    2. Add a text file, an html file, and an image file to that repo and commit your changes.
    3. Add a new text file and change the previously added text file. Commit you changes.
    4. Create a branch named feature1. Change to that branch and make a change to the html file. Commit your changes.
    5. Switch back to the master branch and make a different change to the html file. Commit your changes.
    6. Merge the feature1 branch into the master branch.
    7. Create a repo on GitHub.com. Push your local repository to GitHub.com.
    Submit the URL to your public repo on GitHub.com on courseworks.
